How To Use

1) Log In

2) Select Query type. E.G: INSERT, DELETE, Q6 or Q27

3) Fill the required forms for that Query

---Example 1: Query: Q27, K:10, Month:1 and Year: 2016

---Example 2: Query: Q3, K:10 and Year: 2016

---Example 3: Query: INSERT, fill the Isert New User form

---Example 4: Query: DELETE, fill the Delete Exisitng User form

4) Press the GO button to get your Query results

*Admin users will only get options to INSERT or DELETE
